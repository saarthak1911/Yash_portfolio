## Project Overview

This project was built as a client deliverable — a fully responsive, modern personal portfolio website to represent Yash Nandurge's professional profile online. The goal was to create a clean, high-impact site that effectively showcases his skills, projects, and experience to potential employers and collaborators.

## Deployment

The website is deployed on **Vercel** and is accessible at the live URL above.

### Local Setup (for development/handover)

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate into the project
cd your-repo-name

# Install dependencies
npm install

# Run development server
npm run dev
```

### Production Build

```bash
npm run build
```
